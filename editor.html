콘텐츠 편집

모바일 청첩장 편집 페이지

이 페이지에서 초대장 초안을 저장하고, 원하면 GitHub Pages 공개 페이지까지 바로 발행할 수 있습니다. `저장`은 이 기기 초안용이고 `공개 발행`은 공유 링크에 반영됩니다.

공개 발행

공유 링크에 바로 반영합니다

편집기는 보호된 서버리스 발행 엔드포인트로만 요청을 보내고, GitHub 토큰은 그 서버 안에만 보관됩니다. 브라우저에는 GitHub 토큰이 내려오지 않습니다.

발행 연결

발행 전 확인

새로 선택한 사진 파일은 `공유 페이지에 발행하기`를 눌러야 실제 사이트에 업로드됩니다.

갤러리 이미지는 캐러셀 카드와 전체 화면 스와이프 뷰어로 표시됩니다.

GitHub 토큰은 서버리스 엔드포인트 비밀값으로만 저장되고, 이 편집 페이지에는 저장되지 않습니다.

RSVP 응답 관리에는 같은 Worker 안의 KV 저장소가 필요합니다.

GitHub Pages 반영에는 보통 1~2분 정도 걸릴 수 있습니다.
